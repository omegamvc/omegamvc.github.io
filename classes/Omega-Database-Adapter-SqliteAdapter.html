\Omega\Database\AdapterSqliteAdapter

Sqlite adapter class.

The SqliteDatabaseAdapter class is an implementation of the abstract AbstractDatabaseAdapter and is specifically tailored for SQLite database connections. This adapter provides SQLite-specific database management features while inheriting the common database functionality defined in the parent class.
